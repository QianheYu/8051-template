# 此目录存放DAC0832芯片的示例程序  

## 基础示例代码  
包括sine-wave（正弦波），square（方波），triangular（三角波），serrated（锯齿波）。  
以上四个示例均使用直通模式。  

## 综合应用
synthesized示例是sine-wave（正弦波），square（方波），triangular（三角波），serrated（锯齿波）的综合示例。 
其中使用P1端口连接开关作为模式切换，同样使用直通模式。  
synthesiszed-double是使用双缓冲模式正弦波、方波示例。
